# Icons Needed

Create PNG icons at these sizes:

- `icon-16.png` - 16x16 pixels (toolbar)
- `icon-48.png` - 48x48 pixels (extension management)
- `icon-128.png` - 128x128 pixels (Chrome Web Store)

## Design Suggestions

- Use the Speddy brand colors (blue: #2563eb)
- Simple "S" logo or book/pencil icon
- Should be recognizable at small sizes

## Tools

- Figma (free)
- Canva (free)
- Adobe Illustrator
- GIMP (free)

## Temporary Solution

For testing without icons, you can:
1. Comment out the icon entries in manifest.json
2. Chrome will use a default puzzle piece icon
